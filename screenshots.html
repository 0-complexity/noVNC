noVNC by kanaka

VNC client using HTML5 (Web Sockets, Canvas) with encryption (wss://) support.

See homepage and README for more info.

noVNC is designed to be easily integrated into other websites so my focus is on function with minimal fluff. So yes, my screenshots are pretty boring. If you have a cool shot of noVNC in action, put it on flickr (or wherever) and let me know and if I like it I'll add it here.
